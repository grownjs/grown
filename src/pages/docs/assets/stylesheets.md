### Stylesheets

&mdash; Same as `.js` files but for LESS, Sass, PostCSS or Styl:

```css
/* app/assets/stylesheets/example.post.css */

::root {
  --color: red;
}

.error {
  color: var(--color);
}
```

Source file is saved to `public/stylesheets/example.css` dest.
